Privacy Policy

Last updated: March 2026  ·  CodeDNA Claude Code Plugin

CodeDNA does not collect, transmit, or store any personal data or code. All processing happens locally on your machine.

1. Overview

CodeDNA is a Claude Code plugin that annotates source files with architectural metadata. It operates entirely within your local development environment and does not communicate with any external server owned or operated by the CodeDNA project.

2. Data we do not collect

  • Source code or file contents
  • Project structure or file paths
  • Personal information of any kind
  • Usage statistics, telemetry, or analytics
  • API keys or credentials

3. How the plugin works

When you run a CodeDNA slash command (e.g. /codedna:init), the plugin instructs Claude Code to read your local files and write annotations back to those same files. This interaction happens entirely within the Claude Code session on your machine.

CodeDNA does not introduce any additional network requests beyond those already made by Claude Code as part of its normal operation with Anthropic's API.

4. Anthropic's privacy policy

Because CodeDNA runs inside Claude Code, your interactions with the AI model are governed by Anthropic's Privacy Policy. Please review it to understand how your prompts and code are handled by the underlying model.

5. Third-party services

CodeDNA does not integrate with or send data to any third-party service. No analytics, crash reporting, or tracking libraries are included.

6. Changes to this policy

If this policy is updated, the revised version will be published at this URL with an updated date at the top. No material changes are anticipated given the local-only nature of the plugin.
